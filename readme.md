# 如何使用

## npm

```shell
npm install yam-cesium
```

```javascript
import * as yamCesium from 'yam-cesium'
```

或者
```javascript
import {Map} from 'yam-cesium'
```

## 浏览器
```javascript
<script src="./dist/build/yam.cesium.min.js"></script>
```
注意要先引入Cesium.js

## 编译

```shell
git clone git@github.com:Flashowing/yamCesium.git
cd yamCesium
npm install
npm run build
```
